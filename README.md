### Installation

1. Python 3.9
2. `python -m venv .venv`
3. `source .venv/bin/activate`
4. `pip install --upgrade pip`
5. `pip install -r requirements.txt -r requirements.dev.txt`
6. in the root dir: `pip install -e .`
